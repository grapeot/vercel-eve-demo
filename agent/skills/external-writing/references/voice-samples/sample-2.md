# Voice Calibration 2: Let Concepts Arrive Through a Process

> Synthetic style sample. Its facts are fictional and must never be reused as evidence.

A research request enters Northstar as one record. Search workers can branch from it, but each worker writes to a different evidence file. The parent reads those files only after the workers finish. This small constraint prevents two agents from silently rewriting the same claim table.

The product calls this workspace isolation. The name matters less than the sequence. First the parent divides the question. Then each worker gathers evidence within a named boundary. Finally the parent compares the results and owns the decision. Readers have already seen the behavior by the time the label appears.

Use this pattern when several abstract components must be compared. Keep one familiar request moving through the system and introduce labels after their effects are visible.
