# Source Policy

Use the strongest available evidence for each claim:

1. Primary sources: official documentation, source code, filings, standards, datasets, and direct announcements.
2. Independent technical evidence: reproducible tests, engineering reports, and named expert analysis with inspectable methods.
3. Reporting and discovery sources: useful for locating events and disagreements, but not enough alone for detailed technical claims.

Search snippets only establish that a page may be relevant. Extract or open the source before relying on a number, quote, limitation, chronology, or mechanism. Keep conflicting sources visible. Never resolve a disagreement by counting links.

For every consequential claim, record the claim, URL, source tier, direct evidence, inference boundary, and confidence. Mark derived conclusions as analysis rather than source statements.
